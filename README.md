# Introduction
Focusing on data analyst roles, this project explores top-paying jobs, in-demand skills, and where high demand meets high salary in data analytics.

SQL queries? Check them out here: [project_sql folder](/project_sql/)


### The questions I wanted to answer through my SQL queries were:

1. What are the top-paying data analyst jobs?
2. What skills are required for these top-paying jobs?
3. What skills are most in demand for data analysts?
4. Which skills are associated with higher salaries?
5. What are the most optimal skills to learn?

# Tools I Used

- **SQL:** Used for querying and analyzing data
- **PostgreSQL:** Database management system
- **Visual Studio Code:** SQL development environment
- **Git & GitHub:** Version control and project management

# Conclusions

### Key Insights

- Data analyst salaries vary significantly across roles.
- SQL is the most in-demand skill and is commonly required for high-paying positions.
- Specialized skills are associated with higher average salaries.
- Combining salary and demand helps identify the most valuable skills to learn.

### Closing Thoughts

This project strengthened my SQL skills and provided insights into the data analyst job market. The results can help prioritize skill development based on both demand and salary.
